# PixelArt Gen 🎨

A browser-based pixel art editor. Pick a color and grid size, click or drag to draw, then export your creation as a PNG.

## What I Learned

1. DOM Manipulation to generate and color grid elements.
2. Custom CSS variables and using them to implement dynamic grids.
3. Using event handlers.
4. Using Canvas API and downloading PNG files.

## Tech Stack

HTML · CSS · Vanilla JavaScript
